# 小说阅读器

一个现代化的小说阅读网站，支持自动识别和解析txt格式的小说文件。

## 功能特点

- 📚 **自动识别小说文件** - 扫描novel文件夹中的txt文件
- 🎨 **现代化界面** - 简约现代的设计风格
- 🌙 **多主题支持** - 日间、夜间、护眼主题
- 📖 **优秀的阅读体验** - 字体调节、行间距设置、书签功能
- 🔍 **全文搜索** - 支持小说名称、作者、章节内容搜索
- 📱 **响应式设计** - 完美适配移动端和桌面端
- 💾 **阅读进度保存** - 自动保存阅读位置和设置
- ⚡ **性能优化** - 分章节加载，支持大文件
- 🚀 **静态部署** - 可部署到GitHub Pages

## 文件格式要求

小说txt文件应遵循以下格式：

```
小说标题
label_author作者名

简介:
小说简介内容，可以有多行
==================================================
第1章 章节标题
章节内容...

第2章 章节标题
章节内容...
```

## 本地运行

1. 克隆或下载项目
2. 将小说txt文件放入`novel`文件夹
3. 使用本地服务器打开`index.html`

```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve .

# 然后访问 http://localhost:8000
```

## 部署到GitHub Pages

1. 将项目推送到GitHub仓库
2. 进入仓库设置页面
3. 找到"Pages"选项
4. 选择"Deploy from a branch"
5. 选择"main"分支和"/ (root)"文件夹
6. 点击"Save"即可部署

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **存储**: LocalStorage
- **部署**: GitHub Pages
- **文件处理**: FileReader API

## 快捷键

在阅读界面中，可以使用以下快捷键：

- `←` - 上一章
- `→` - 下一章
- `B` - 添加/移除书签
- `F` - 全屏模式
- `ESC` - 退出全屏或返回列表
